* **#OOTD는 쇼핑몰과 커뮤니티를 결합한 웹애플리케이션 프로젝트입니다.**
* **기본적으로 회원과 관련된 기능들이 서로 연관되어 CRUD기능이 동작하도록 DB를 설계하였습니다.**
